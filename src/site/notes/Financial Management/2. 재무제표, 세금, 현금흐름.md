# 2. 재무제표, 세금, 현금흐름

## 2.1 재무상태표 (Balance Sheet)

- **정의**: 특정 시점에 기업의 자산·부채·자본 상태
- **항등식**: `자산 = 부채 + 자본`
- 유동자산을 위에, 유동부채를 위에 작성
- **장부가치 vs 시장가치**
  - 장부가치: 원가 또는 공정가치평가모형 (시장 거래가치 아님)
  - 시장가치: 실제 시장에서 매매될 수 있는 가격
  - ⭐ 재무관리 관점에서는 **시장가치**가 더 중요 → 목표: 자기자본의 현재 시장가치 극대화

### 순운전자본 (Net Working Capital, NWC)
- `NWC = 유동자산 - 유동부채`
- 건실한 기업은 통상 양수(+)
- 유동성 ↑ → 재무적 곤경 가능성 ↓ (but 수익성과 trade-off)

---

## 2.2 포괄손익계산서 (Income Statement)

- **정의**: 특정 기간의 성과 측정
- **흐름**: 매출액 → (비용 차감) → 당기순이익(NI, Net Income)
- **회계이익 ≠ 현금흐름**
  - 발생주의 회계: 현금 수령 시점이 아닌 판매 시점에 수익 인식
  - 비현금항목 존재: 대표 예시 = **감가상각비** (비용 처리되지만 현금 유출 없음)

## 2.3 세금

### 한국 법인세율 (2026년 현재, 누진세율)
| 과세표준 | 세율 |
|---|---|
| 2억원 이하 | 9% |
| 2억 ~ 200억 | 19% |
| 200억 ~ 3,000억 | 21% |
| 3,000억 초과 | 24% |

### 평균세율 vs 한계세율
- **평균세율**: 납부한 총 세금 ÷ 총 과세표준
- **한계세율**: 1원을 추가로 벌었을 때 내는 세율 (= 현재 속한 세율 구간)
- 예시 (미국, 개인소득 $100,000)
  - 총 세금: $18,021
  - 평균세율: 18.02%
  - 한계세율: 24% (해당 구간)

## 2.4 현금흐름

### 현금흐름 항등식
```
자산의 현금흐름 = 채권자의 현금흐름 + 주주의 현금흐름
```

### 자산의 현금흐름
```
자산의 CF = 영업현금흐름(OCF) - 순자본지출 - NWC의 변동
OCF = EBIT + 감가상각비 - 법인세
순자본지출 = 기말 순비유동자산 - 기초 순비유동자산 + 감가상각비
NWC 변동 = 기말NWC - 기초NWC
```

### 채권자의 현금흐름
```
채권자의 CF = 이자비용 - 순신규차입금
```

### 주주의 현금흐름
```
주주의 CF = 배당지급액 - 순신규주식발행액
```

### EBIT / EBITDA
- **EBIT** = Earnings Before Interest and Taxes = 이자 및 세금 차감 전 이익 (영업이익)
- **EBITDA** = Earnings Before Interest, Taxes, Depreciation and Amortization
  - EBIT + 감가상각비(D) + 무형자산상각비(A)
  - 감가상각비는 비현금항목 → EBITDA는 현금창출능력에 더 가까운 지표
  - 실무에서 기업가치 평가 시 많이 활용
