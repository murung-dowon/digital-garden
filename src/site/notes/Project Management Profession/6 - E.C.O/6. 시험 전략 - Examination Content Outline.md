# 1. ECO란?

**Examination Content Outline** — PMI가 공식 발행하는 **PMP 시험 청사진**.
"어떤 영역에서 몇 % 출제하는지" 정의한 문서. 시험 문제 해설에 "ECO Process 6.3" 같이 표기됨.

시험의 **50%는 예측형(Predictive)**, **50%는 애자일/혼합형(Agile/Hybrid)** 방식 문제.

---

# 2. 3개 도메인 & 출제 비율

| 도메인 | 비율 | 핵심 키워드 |
|--------|------|------------|
| **People** | 42% | 팀 관리, 리더십, 이해관계자, 갈등 해결 |
| **Process** | 50% | 일정/원가/범위/리스크/조달 등 기술적 관리 |
| **Business Environment** | 8% | 전략 정렬, 컴플라이언스, 조직 변화 |

> 📝 **시험 포인트:** Process가 50%로 제일 많지만, People(42%)도 거의 절반. "기술적 답" vs "리더십 답" 구분이 핵심.

# 6. 시험 포인트 요약

- **"ECO Process 6.3"** → 일정 관리 Task 6, 네트워크 다이어그램 관련 문제 → **PDM(AON)** 답
- **People 42%** → 리더십/팀/이해관계자 문제에서 "PM이 어떻게 행동해야 하나" 유형
- **Process 50%** → 기술적 계획·관리 프로세스 관련 문제
- **"어떤 접근방식을 써야 하나"** 문제 → 개발 접근방식(Predictive/Agile/Hybrid) 먼저 파악 후 답 선택
